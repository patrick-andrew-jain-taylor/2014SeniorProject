This is a library for the SEPS525-based OLED shield.

http://www.tablix.org/~avian/blog/articles/arduino_oled_shield/

Written by Tomaz Solc, based on the Adafruit GFX library.

Place the contents of this repository into a "SEPS525_OLED" folder under your
<arduinosketchfolder>/libraries/ folder. You may need to create the libraries
subfolder if its your first library. Restart the IDE

Also requires the Adafruit_GFX library for Arduino.
https://github.com/adafruit/Adafruit-GFX-Library
